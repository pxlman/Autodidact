The Learning Loop

Query comes in → 🧠 Brain thinks first, checks memory → Confident? → 🟢 YES: Answer locally ($0) → 🔴 NO: Ask cloud model → Extract knowledge from response → Store in local memory → Next time → answer locally

Thompson Sampling — agent learns which signals to trust, no manual tuning

Ebbinghaus Decay — frequently-used knowledge sticks, stale knowledge fades
